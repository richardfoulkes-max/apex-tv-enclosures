Design Language Specifications

Geometry & Proportions

Element Specification Rationale
Corner Radius (External) 12mm Consistent across all products. Soft but not childish.
Corner Radius (Internal/Details) 6mm Half of external for visual hierarchy
Panel Gap / Shadow Line 3mm Visible intentional gap creates depth perception
Frame Depth 25mm Creates substantial "picture frame" effect
Panel Recess 5mm Panels sit behind frame, creating shadow
Minimum Touch Target 44mm Comfortable finger operation (Apple HIG standard)
Logo Size 40mm width Subtle, confident. Embossed, not applied.
