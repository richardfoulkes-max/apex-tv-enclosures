## 1. Product Overview

### 1.1 Purpose
Modular outdoor enclosure system for residential pool equipment (pumps, filters, chlorinators, automation). Primary function is **acoustic noise reduction** (70-80%) while providing weather protection and aesthetic improvement.

### 1.2 Design Philosophy
- **Modular system:** Base unit + optional extension modules
- **Tool-free assembly:** Tongue-and-groove connections
- **Passive ventilation:** No fans required (unlike TV enclosures)
- **Acoustic focus:** 25-50mm closed-cell foam lining

### 1.3 Target Market
UAE/GCC residential pool installations. Climate: 45-55°C ambient, high humidity, salt-laden coastal air.

## 3. Materials Specification

### 3.1 Primary Structure

| Component | Material | Specification |
|-----------|----------|---------------|
| Frame panels | 5052-H32 Aluminum | 3.0mm thickness |
| Louvers | 5052-H32 Aluminum | 2.0mm thickness |
| Corner brackets | 5052-H32 Aluminum | 3.0mm, welded |

**Why 5052-H32:**
- Superior corrosion resistance vs 6061
- Salt spray resistance for coastal installations
- Good weldability and formability
- Standard marine-grade aluminum

### 3.2 Hardware (316 Stainless Steel Required)

| Component | Specification | Quantity per BASE |
|-----------|---------------|-------------------|
| Machine screws | M5 × 12mm hex socket | 24 |
| Self-tapping screws | M4 × 10mm | 48 |
| Nuts | M5 nyloc | 24 |
| Washers | M5 flat | 48 |
| Piano hinge | 316 SS, 600mm length | 1 |
| Quick-release latches | 316 SS, compression type | 2 |
| Gas struts | 100N, 316 SS | 2 |

**CRITICAL:** All fasteners MUST be 316 stainless steel. Many pool installations use salt chlorinators which create a corrosive environment. 304 stainless will fail within 12-18 months.

### 3.3 Acoustic Foam

| Parameter | Specification |
|-----------|---------------|
| Type | Closed-cell polyethylene |
| Density | 30-40 kg/m³ |
| Thickness | 25mm (sides/top), 50mm (rear) |
| NRC Rating | 0.65-0.75 |
| Fire Rating | Class O / ASTM E84 Class A |
| Adhesive | Pressure-sensitive acrylic |
| Installation | Factory-bonded to interior panels |

**Why Closed-Cell:**
- Does NOT absorb water (critical for humid environment)
- Resists mold and mildew
- Maintains acoustic properties when wet
- Chemical resistant (chlorine, salt)

## 4. Ventilation Design

### 4.1 Passive Convection System

Unlike TV enclosures which require active fans, pool equipment enclosures use **passive convection ventilation**:

```
Cool air enters through bottom louvers
    ↓
Air warms as it passes over pump motor (~200-500W heat)
    ↓
Warm air rises naturally (convection)
    ↓
Exits through top louvers
```

### 4.2 Louver Specifications

| Parameter | Value |
|-----------|-------|
| Bottom intake | 500 × 40mm opening |
| Top exhaust | 500 × 40mm opening |
| Louver angle | 45° downward (rain rejection) |
| Louver spacing | 15mm between slats |
| Effective open area | 40-50% |

### 4.3 Acoustic Baffles

Inside both intake and exhaust louvers, install acoustic baffles:
- Material: Same closed-cell foam as walls
- Configuration: Labyrinth path (sound cannot travel straight through)
- Effect: Blocks direct sound transmission while allowing airflow

## 5. Modular Connection System

### 5.1 Tongue-and-Groove Design

Modules connect via interlocking tongue-and-groove joints:

```
┌─────────────────┐    ┌─────────────────┐
│                 │▓▓▓▓│                 │
│    Module A     │▓▓▓▓│    Module B     │
│                 │▓▓▓▓│                 │
└─────────────────┘    └─────────────────┘
                  ▓▓▓▓ = Tongue + EPDM seal
```

### 5.2 Connection Points

| Location | Type | Notes |
|----------|------|-------|
| Top edge | Tongue (male) | On all modules |
| Bottom edge | Groove (female) | Receives tongue from above |
| Left edge | Groove (female) | Receives next module |
| Right edge | Tongue (male) | Connects to next module |

### 5.3 Assembly Sequence

1. Place BASE module on ground pad
2. Slide EXT-A onto BASE (right edge)
3. Press firmly until click (EPDM compresses)
4. Continue with additional modules
5. **No tools required**

## 8. Chiller Module - Special Requirements

### 8.1 Why Chiller is Different

Pool chillers contain compressors that require significant airflow for heat rejection. **Cannot be fully enclosed** like pump/filter modules.

### 8.2 APE-CHILL Design

| Feature | Specification |
|---------|---------------|
| Front panel | Fully louvered (50% open) |
| Side panels | Fully louvered (50% open) |
| Rear panel | Solid with acoustic foam |
| Top panel | Solid (rain protection) |
| Louver angle | 45° horizontal |
| Noise reduction | 50-60% only (vs 70-80% for enclosed modules) |

### 8.3 Acoustic Baffles

- Install behind all louvered panels
- Use 25mm foam in labyrinth configuration
- Reduces direct sound transmission
- Does NOT significantly restrict airflow
