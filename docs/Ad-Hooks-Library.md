## How to Use This Document

1. **Test 5-10 hooks per day** with $20 each
2. **Kill losers after 48 hours** (CTR <1%, no conversions)
3. **Scale winners** (CTR >2%, conversions coming in)
4. **Iterate** - create variations of winning hooks

**Ad Structure**:
- **Hook** = First line (stops the scroll)
- **Primary Text** = Body copy (builds desire)
- **CTA** = Call to action

## Winner Iteration Template

When you find a winning hook, create 5 variations:

1. **Same hook, different image** (lifestyle vs product shot)
2. **Same hook, shorter copy** (remove middle paragraph)
3. **Same hook, longer copy** (add testimonial)
4. **Same hook, different CTA** (Reserve Now vs Learn More)
5. **Same hook, different audience** (pool owners vs outdoor kitchen builders)
