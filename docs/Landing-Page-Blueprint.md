## Color & Typography Guidelines

### Color Palette (Premium Outdoor Feel)

| Element | Color | Hex | Rationale |
|---------|-------|-----|-----------|
| Primary | Deep Navy | #1a1a2e | Luxury, tech |
| Accent | Gold/Amber | #d4a574 | Premium, warmth |
| CTA Button | Vibrant Orange | #e85d04 | Urgency, action |
| Background | Off-white | #fafafa | Clean, spacious |
| Text | Charcoal | #2d2d2d | Readable |

### Typography

- **Headlines:** Bold sans-serif (Inter, Montserrat)
- **Body:** Clean sans-serif (Inter, Open Sans)
- **Accent:** Consider subtle serif for testimonials

## Mobile-First Requirements

Based on research: **70%+ of traffic is mobile**

| Element | Mobile Requirement |
|---------|-------------------|
| Hero image | Full-width, not cropped |
| CTA button | Sticky bottom bar OR large touch target |
| Navigation | Hamburger menu |
| Images | Lazy load, compressed |
| Forms | Minimal fields, large inputs |
| Load time | <3 seconds |
